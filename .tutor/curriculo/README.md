# Currículo

Esta pasta contém o roadmap dividido em 14 módulos (00 a 13), cada um em um arquivo dedicado. A IA deve ler o arquivo do módulo correspondente antes de criar ou revisar qualquer aula daquele módulo.

Veja o resumo de navegação em [mapa-geral.md](mapa-geral.md).

Cada arquivo de módulo contém: objetivo, pré-requisitos, aulas, atividades, exercícios esperados, critérios para avançar, e como a IA deve (e não deve) ajudar. Dos Módulos 00 a 06, os exemplos são só em Python — TypeScript entra a partir do Módulo 07.
