# Módulo 01 — Lógica e pensamento computacional (Python)

**Duração sugerida:** 3 a 4 semanas (9 aulas)

## Objetivo do módulo

Aprender a resolver problemas pequenos usando Python, antes de se preocupar com qualquer framework ou com uma segunda linguagem.

## Pré-requisitos

Módulo 00 concluído.

## Aulas

### Aula 1 — Entrada, processamento e saída

`input()`, `print()`, e a ideia de um programa como uma sequência de passos.

### Aula 2 — Variáveis

Nomear e guardar valores, tipos básicos (`int`, `float`, `str`, `bool`).

### Aula 3 — Operadores aritméticos e lógicos

`+ - * / // % **`, comparações, `and`/`or`/`not`.

### Aula 4 — Condicionais

`if`/`elif`/`else`, condições compostas.

### Aula 5 — Laços

`for`, `while`, `break`, `continue`.

### Aula 6 — Funções

Definir funções, parâmetros, retorno, escopo.

### Aula 7 — Decomposição de problemas

Quebrar um problema grande em funções pequenas.

### Aula 8 — Validação básica de dados

Verificar entradas inválidas antes de processar.

### Aula 9 — Leitura de mensagens de erro

Entender tracebacks do Python e usá-los para depurar sozinho.

## Atividades / exercícios esperados

Resolver cada exercício em Python e explicar em texto como o algoritmo funciona. Exercícios recomendados:

- calculadora simples;
- conversor de temperatura;
- cálculo de IMC;
- verificador de número par ou ímpar;
- contador de vogais;
- tabuada;
- soma de itens de uma lista;
- encontrar o maior número;
- validar senha;
- simular caixa eletrônico simples.

## Critérios para avançar

Resolver problemas pequenos sem pedir a solução pronta para a IA.

- [ ] Resolve pelo menos 6 dos 10 exercícios recomendados sem ajuda direta de solução.
- [ ] Consegue explicar, em texto, como cada algoritmo funciona.
- [ ] Reconhece e corrige erros de lógica sozinho na maioria das tentativas.

## Como a IA deve ajudar neste módulo

- Fazer perguntas que ajudem o aluno a chegar à solução.
- Se o raciocínio estiver errado, explicar o conceito e propor um exercício parecido mais simples.
- Prompt de referência para o aluno usar:

```text
Estou aprendendo lógica de programação em Python. Não resolva o exercício por mim.
Me faça perguntas para eu chegar à solução. Se meu raciocínio estiver errado,
explique o conceito e me dê um exercício parecido mais simples.
```

## Como a IA NÃO deve ajudar neste módulo

- Não entregue a solução do exercício antes do aluno tentar pelo menos uma vez.
- Não use jargão de frameworks, bibliotecas avançadas ou de outras linguagens — o foco aqui é lógica pura em Python.
