# Roadmap — CodeMentor BR

Trilha principal: **Node.js com TypeScript + Python**, com foco em fundamentos de programação e, mais adiante, no modelo **Spec-Driven Developer**.

> Este arquivo é um resumo de navegação. O conteúdo completo de cada fase — com exercícios recomendados, projetos e critérios para avançar — está em `.tutor/curriculo/`. A fonte original deste roadmap (com explicações mais longas) está em `roadmap.md`, na raiz do repositório.

## A tese central

O diferencial de um programador não é mais só "saber digitar código" — IA já faz isso rápido. O diferencial é entender o problema antes da solução, modelar dados e regras, avaliar se o código gerado é bom, escrever especificações claras e usar IA como ferramenta, não como muleta.

## Regras de ouro para estudar na era da IA

1. **No início, faça no modo hard** — tente implementar manualmente antes de pedir ajuda da IA.
2. **Código gerado não é código aprovado** — todo código de IA passa por triagem humana.
3. **Fundamentos antes de frameworks** — frameworks mudam, lógica e estrutura de dados permanecem.

Veja a explicação completa dessas regras em `roadmap.md`, seção 4.

## Checklist de maturidade

- **Nível 1 — Iniciante real**: entende variáveis, funções e condicionais; resolve exercícios pequenos; usa Git básico; não depende da IA para tudo.
- **Nível 2 — Iniciante consistente**: cria projetos pequenos; usa tipos básicos; escreve README; compara soluções em Python e TypeScript.
- **Nível 3 — Pré-júnior**: cria API com banco; escreve testes; usa migrations; valida entrada; trata erros.
- **Nível 4 — Júnior com base forte**: separa regra de negócio de infraestrutura; revisa código gerado por IA; escreve specs; quebra features em tasks.
