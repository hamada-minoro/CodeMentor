# API Financeira

Projeto do Módulo 08/09 (veja `.tutor/curriculo/modulo-08-backend.md` e `.tutor/curriculo/modulo-09-banco-de-dados.md`), também recomendado para o portfólio no Módulo 13.

## Objetivo

Mostrar backend real: banco de dados, autenticação, testes e documentação.

## Stack sugerida

- Versão 1: Node.js + TypeScript + Fastify ou NestJS + PostgreSQL + Prisma.
- Versão 2: Python + FastAPI + Pydantic + SQLAlchemy.

## Funcionalidades esperadas

- Cadastro de usuário e login
- Cadastro de categorias
- Receitas e despesas
- Listagem por período
- Resumo mensal e saldo
- Exportação CSV
- Testes básicos

## O que documentar no seu projeto

Diagrama simples de entidades, endpoints, exemplos de request/response, regras de negócio, erros esperados, como rodar localmente e como rodar os testes.

## Sobre este diretório

Este README descreve o propósito do projeto, não a implementação — construa a sua própria versão.
