# CLI de Tarefas

Projeto de portfólio sugerido na Fase 12 (veja `.tutor/curriculo/fase-12-portfolio.md`) e introduzido como projeto da Fase 02 (`.tutor/curriculo/fase-02-typescript-python.md`).

## Objetivo

Mostrar lógica, manipulação de arquivos, estrutura de dados e organização simples.

## Stack sugerida

- Python
- TypeScript

## Funcionalidades esperadas

- Criar tarefa
- Listar tarefas
- Marcar como concluída
- Remover tarefa
- Salvar em arquivo JSON
- Carregar do arquivo ao iniciar

## Sobre este diretório

Este README descreve o propósito do projeto, não a implementação. Construa sua própria versão em `workspace-aluno/` ou em um repositório próprio, e documente aqui (ou em um README equivalente no seu projeto) como rodar, quais estruturas de dados você usou e quais erros encontrou.
