# Módulo 00 — Ambiente, terminal, Git e rotina de estudos

**Duração sugerida:** 1 semana (5 aulas)

## Objetivo do módulo

Deixar a máquina pronta para estudar e versionar código, e criar o hábito de registrar aprendizado.

## Pré-requisitos

Nenhum. Este é o módulo inicial.

## O que instalar

- VS Code
- Git
- Python 3.11+ e pip (ou uv)
- Node.js LTS e um gerenciador de pacotes (npm, pnpm ou yarn) — instale agora, mas só vai usar de verdade no Módulo 07
- Opcional neste módulo: Docker, PostgreSQL, SQLite (necessários a partir do Módulo 09)

## Aulas

### Aula 1 — Terminal básico

Navegar em pastas e arquivos, criar e mover arquivos, comandos essenciais (`cd`, `ls`/`dir`, `mkdir`, `rm`, `cat`).

### Aula 2 — Git e GitHub

Inicializar um repositório Git, fazer commits, criar conta no GitHub e subir um repositório.

### Aula 3 — README e documentação básica

Criar e editar um `README.md` simples, explicando um projeto.

### Aula 4 — VS Code e Python

Abrir e organizar um projeto no VS Code (`code .`), instalar a extensão de Python, rodar um script `.py` pelo terminal integrado.

### Aula 5 — Rotina de estudo

Criar o diário de aprendizado e entender o ciclo de estudo deste repositório (`.tutor/fluxo-de-estudo.md`).

## Atividades / exercícios esperados

Criar um repositório de prática com a estrutura:

```text
fundamentos/
  python/
projetos/
docs/
  diario-de-aprendizado.md
```

Fazer ao menos 5 commits pequenos e descritivos.

## Critérios para avançar

- [ ] Consegue navegar em pastas e arquivos pelo terminal.
- [ ] Consegue inicializar um repositório Git e fazer commit.
- [ ] Tem uma conta no GitHub com um repositório criado.
- [ ] Sabe abrir um projeto no VS Code pelo terminal (`code .`) e rodar um script Python.

## Como a IA deve ajudar neste módulo

- Explicar comandos de terminal e Git sob demanda, com exemplos pequenos.
- Ajudar a resolver erros de instalação, perguntando o que já foi tentado.

## Como a IA NÃO deve ajudar neste módulo

- Não instale nada sozinha "para economizar tempo" — o aluno precisa passar pelo processo de instalação manual ao menos uma vez.
- Não gere o conteúdo do `README.md` ou do diário por inteiro — apenas oriente o que incluir.
