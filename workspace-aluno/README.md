# `workspace-aluno/`

Este é o seu espaço livre — para criar códigos, testes e estudos fora do ciclo formal de tópicos.

## Como usar

- Use `workspace-aluno/python/` para códigos em Python.
- Use `workspace-aluno/typescript/` para códigos em TypeScript.
- Exercícios gerados pelo tutor devem ficar preferencialmente em `estudos/`, não aqui.
- Não misture gabaritos com suas próprias respostas.
- Mantenha nomes de arquivo claros (`calculadora_imc.py`, não `teste2.py`).
- Faça commits pequenos e frequentes.
- Se um projeto aqui crescer e ficar relevante para portfólio, escreva um README para ele e considere movê-lo para `exemplos-projetos/` ou para um repositório próprio.
