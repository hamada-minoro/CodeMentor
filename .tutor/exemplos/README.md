# Exemplos

Esta pasta contém exemplos funcionais de como o ciclo de estudo do tutor se materializa em arquivos reais.

- `exemplo-ciclo-topico-logica/` — simula o tópico "Variáveis, entrada, processamento e saída" da Fase 1, mostrando como ficam `README.md`, `exercicios.md`, `revisao.md`, `nota.md` e `consolidacao.md` depois de um ciclo completo.

Use este exemplo como referência de formato — não como gabarito a ser copiado para seus próprios tópicos.
