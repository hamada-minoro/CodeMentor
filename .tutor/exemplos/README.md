# Exemplos

Esta pasta contém exemplos funcionais de como o ciclo de estudo do tutor se materializa em arquivos reais.

- `exemplo-ciclo-aula-logica/` — simula a aula "Variáveis, entrada, processamento e saída" do Módulo 01, mostrando como ficam `README.md`, `exercicios.md`, `revisao.md`, `nota.md` e `consolidacao.md` depois de um ciclo completo.

Use este exemplo como referência de formato — não como gabarito a ser copiado para suas próprias aulas.
