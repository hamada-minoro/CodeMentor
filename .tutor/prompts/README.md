# Prompts

Prompts prontos para você copiar e colar na sua IA — Claude Code, GitHub Copilot Chat, Continue.dev ou IA local.

| Arquivo | Para quê |
|---|---|
| `iniciar-aula.md` | Começar uma nova aula do roadmap |
| `explicar-conceito.md` | Pedir explicação de um conceito sem resolver exercício |
| `gerar-exercicios.md` | Gerar exercícios adicionais de uma aula |
| `revisar-exercicios.md` | Pedir revisão das suas respostas |
| `corrigir-com-nota.md` | Pedir nota com justificativa baseada em rubrica |
| `gerar-consolidacao.md` | Pedir um exercício de reforço |
| `tirar-duvida-sem-resolver.md` | Tirar dúvida sobre um erro sem receber a solução direta |
| `analisar-erro.md` | Analisar uma mensagem de erro |
| `avancar-proxima-aula.md` | Saber se está pronto para avançar |
| `criar-feature-spec-driven.md` | Iniciar uma feature com o fluxo Spec-Driven Development |

Se você já tem alguma experiência prévia e não sabe em qual módulo começar, use a skill `nivelar-aluno` em vez de um prompt fixo — ela aplica um questionário curto e recomenda o ponto de partida.

Todos os prompts assumem que a IA vai ler `.tutor/tutor-manifest.md` antes de responder.
