# Rubricas

Critérios objetivos de avaliação usados pela IA ao dar nota e revisar trabalho do aluno.

- `escala-de-notas.md` — faixas de 0 a 10 e o que cada uma significa.
- `rubrica-codigo.md` — rubrica de 10 pontos para exercícios e projetos de código.
- `rubrica-logica.md` — rubrica focada em exercícios de lógica (Módulo 01).
- `rubrica-documentacao.md` — rubrica para READMEs e documentação de projeto.
- `rubrica-spec-driven.md` — rubrica para specs, plans e reviews no fluxo Spec-Driven Development.

A IA deve sempre justificar a nota dada, item a item — nunca dar um número sem explicação.
