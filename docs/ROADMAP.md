# Roadmap — CodeMentor BR

Trilha principal: **Python primeiro**, com fundamentos sólidos de programação antes de qualquer segunda linguagem. **Node.js com TypeScript** entra como módulo dedicado (Módulo 07), e mais adiante chega o modelo **Spec-Driven Developer**.

> Resumo de navegação. O conteúdo completo de cada módulo — aulas, exercícios e critérios para avançar — está em [.tutor/curriculo/mapa-geral.md](../.tutor/curriculo/mapa-geral.md).

## A tese central

O diferencial de um programador não é mais só "saber digitar código" — IA já faz isso rápido. O diferencial é entender o problema antes da solução, modelar dados e regras, avaliar se o código gerado é bom, escrever especificações claras e usar IA como ferramenta, não como muleta.

## Por que Python primeiro

Misturar duas linguagens desde o primeiro dia confunde quem está começando: o aluno não sabe se o erro é de lógica ou de sintaxe de uma linguagem que ainda não domina. Por isso, dos Módulos 00 a 06 o currículo é **só em Python**. TypeScript e Node.js entram no Módulo 07 — criar um projeto Node do zero, instalar dependências de TypeScript e automatizar esse scaffolding com IA a partir de um `spec.md` bem descrito.

## Regras de ouro para estudar na era da IA

1. **No início, faça no modo hard** — tente implementar manualmente antes de pedir ajuda da IA.
2. **Código gerado não é código aprovado** — todo código de IA passa por triagem humana: resolve o problema real? cobre erros? tem teste? é simples o bastante?
3. **Fundamentos antes de frameworks** — frameworks mudam, lógica, estrutura de dados, tipagem, arquitetura e testes permanecem.

## Checklist de maturidade

- **Nível 1 — Iniciante real**: entende variáveis, funções e condicionais em Python; resolve exercícios pequenos; usa Git básico.
- **Nível 2 — Iniciante consistente**: cria projetos pequenos em Python com tipos básicos; escreve README; depois do Módulo 07, já compara soluções em Python e TypeScript.
- **Nível 3 — Pré-júnior**: cria API com banco; escreve testes; usa migrations; valida entrada; trata erros.
- **Nível 4 — Júnior com base forte**: separa regra de negócio de infraestrutura; revisa código gerado por IA; escreve specs; quebra features em tasks.

## Depois da base

Depois de concluir o Módulo 13, escolha um foco estratégico:

- **Backend** — NestJS, FastAPI, PostgreSQL avançado, Redis, filas, mensageria, Docker, observabilidade.
- **Dados e automação** — Pandas, SQL, ETL, automações, notebooks, pipelines, integrações com IA.
- **IA aplicada a software** — RAG, embeddings, agentes, MCP, avaliação de respostas, ferramentas de coding agent.
- **Arquitetura** — Clean Architecture, DDD, arquitetura hexagonal, event-driven architecture, escalabilidade.
