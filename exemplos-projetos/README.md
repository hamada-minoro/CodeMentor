# `exemplos-projetos/`

Projetos de referência do portfólio recomendado (veja `.tutor/curriculo/fase-12-portfolio.md`). Estes diretórios contêm propósito, escopo e — no caso do projeto spec-driven — documentação de exemplo, mas **não código de referência completo**. A ideia é que você construa; copiar uma implementação pronta destruiria o ponto pedagógico.

- `cli-tarefas/` — CLI de tarefas em Python e TypeScript.
- `api-financeira/` — API de controle financeiro pessoal.
- `projeto-spec-driven/` — exemplo de projeto documentado com o fluxo Spec-Driven Development completo (`spec.md`, `plan.md`, `tasks.md`, `tests.md`, `review.md`).
