# Exercícios — Variáveis, Entrada, Processamento e Saída

> Não use IA para gerar a solução destes exercícios. Tente primeiro. Se travar, peça à IA para explicar o conceito, não para resolver.

## Formato esperado das respostas

Coloque suas respostas em:

```text
minhas-respostas/python/exercicio_01.py
minhas-respostas/python/exercicio_02.py
minhas-respostas/typescript/exercicio_01.ts
minhas-respostas/typescript/exercicio_02.ts
```

## Aquecimento

1. Crie uma variável com seu nome e imprima uma frase de saudação usando ela.
2. Crie duas variáveis numéricas e imprima a soma das duas.

## Principais

3. Peça (ou simule, em TypeScript) a entrada de dois números e imprima a soma, a subtração, a multiplicação e a divisão entre eles.
4. Crie um conversor simples de Celsius para Fahrenheit: `F = C * 9/5 + 32`.
5. Peça o nome e a idade de uma pessoa e imprima uma frase dizendo quantos anos ela terá em 10 anos.

## Desafio extra

6. Crie um pequeno "cálculo de troco": dado o valor de uma compra e o valor pago, calcule e imprima o troco. Trate o caso em que o valor pago é menor que o valor da compra (o que deveria acontecer nesse caso?).

## Critérios de avaliação

- O programa lê e usa variáveis corretamente.
- O cálculo está correto.
- A saída é clara e legível.
- O código tem nomes de variável que fazem sentido.
- (Desafio) o caso de erro foi pensado, mesmo que de forma simples.
