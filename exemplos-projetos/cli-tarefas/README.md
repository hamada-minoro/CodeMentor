# CLI de Tarefas

Projeto de portfólio sugerido no Módulo 13 (veja `.tutor/curriculo/modulo-13-portfolio.md`) e introduzido em Python como projeto do Módulo 02 (`.tutor/curriculo/modulo-02-fundamentos-python.md`); a versão em TypeScript fica para depois do Módulo 07.

## Objetivo

Mostrar lógica, manipulação de arquivos, estrutura de dados e organização simples.

## Stack sugerida

- Python
- TypeScript

## Funcionalidades esperadas

- Criar tarefa
- Listar tarefas
- Marcar como concluída
- Remover tarefa
- Salvar em arquivo JSON
- Carregar do arquivo ao iniciar

## Sobre este diretório

Este README descreve o propósito do projeto, não a implementação. Construa sua própria versão dentro da pasta do módulo em `estudos/` ou em um repositório próprio, e documente aqui (ou em um README equivalente no seu projeto) como rodar, quais estruturas de dados você usou e quais erros encontrou.
