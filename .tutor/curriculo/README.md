# Currículo

Esta pasta contém o roadmap dividido em 13 fases, cada uma em um arquivo dedicado. A IA deve ler o arquivo da fase correspondente antes de criar ou revisar qualquer tópico daquela fase.

Veja o resumo de navegação em [mapa-geral.md](mapa-geral.md).

Cada arquivo de fase contém: objetivo, pré-requisitos, tópicos, ordem sugerida, atividades, exercícios esperados, critérios para avançar, exemplos em Python e TypeScript quando fizer sentido, e como a IA deve (e não deve) ajudar.
