# Escala de Notas

| Faixa | Significado |
|---|---|
| 0–2 | Não entendeu o conceito básico. |
| 3–4 | Entende parcialmente, mas não consegue aplicar sozinho. |
| 5–6 | Resolve casos simples, mas com erros conceituais ou muita dependência. |
| 7–8 | Resolve bem, explica o raciocínio e comete poucos erros. |
| 9–10 | Resolve, explica, testa, documenta e consegue adaptar o conceito. |

## Como usar

- Use esta escala junto com a rubrica de 10 pontos em `rubrica-codigo.md` para justificar a nota.
- Nunca atribua uma faixa sem relacionar a pontos concretos observados no código ou na explicação do aluno.
- Notas abaixo de 7 geralmente justificam uma consolidação antes de avançar — mas a decisão final é do aluno.
